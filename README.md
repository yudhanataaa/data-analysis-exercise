#  Computational Biology & Data Analysis Exercise

## 📌 Project Overview
This repository contains a collection of Python-based workflows demonstrating foundational skills in **Data Science, Exploratory Data Analysis (EDA), and Basic Bioinformatics**. The notebook serves as a toolkit for processing biological sequences, cleaning messy datasets, and visualizing scientific data using modern computational libraries.

## 🧰 Tech Stack & Libraries
- **Python 3** (Core logic, algorithmic structures, and sequence manipulation)
- **Pandas** (Data wrangling, missing value imputation, and dataset aggregation)
- **Matplotlib & Seaborn** (Data visualization and statistical distribution plots)

## 🔬 Key Features & Workflows Included

### 1. Biological Sequence Processing (Mini Bioinformatics Toolkit)
- **DNA to Protein Translator:** Developed algorithmic loops and dictionary-mapping to transcribe DNA into RNA and translate RNA codons into Amino Acid sequences.
- **Sequence Normalization & Calculation:** Algorithms to clean messy string inputs and calculate base metrics (e.g., Cytosine content profiling).

### 2. Data Cleaning & Imputation (Public Health Data)
- Conducted diagnoses on missing data (Null values) within global health datasets (e.g., WHO Life Expectancy).
- Implemented quantitative approaches to impute missing values (using mean interpolation) and formatted data for downstream statistical analysis.

### 3. Exploratory Data Analysis & Visualization
- **Population Dynamics:** Filtered and sorted synthetic species population data to identify critically endangered groups using logical subsets.
- **Statistical Visualization:** Utilized Seaborn and Matplotlib to map multi-variable analyses, including:
  - Scatter plots and regression models for health variables.
  - Histogram and Kernel Density Estimation (KDE) to analyze pest population variance across different environmental zones.
  - Boxplots for category-based health and biological metric comparisons.
